# 🚀 Quick Start Guide - LocalBazaar

Get your LocalBazaar app running in **5 minutes**!

## ⚡ Super Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev

# 3. Open browser (auto-opens to http://localhost:3000)
# Done! 🎉
```

## 📱 Using the App

### 1️⃣ First Time Setup (30 seconds)

1. **Login Screen appears**
   - Choose your user type: Customer / Vendor / Delivery Partner
   - Enter any 10-digit phone number (demo mode)
   - Click Continue

2. **Select City** (for Customers only)
   - Choose from 12+ cities (e.g., Mumbai, Delhi, Bangalore)
   - Or search for your city

3. **Start Exploring!**

---

## 👤 As a Customer

### Browse and Shop
```
Home → Click Vendor → Add Products → Cart → Checkout → Track Order
```

**Try This:**
1. Browse vendors by category (Kirana, Medical, Electronics, etc.)
2. Use search bar to find specific shops
3. Click ❤️ to add to favorites
4. Click on any vendor card

**In Vendor Shop:**
1. See all products with prices
2. Click "Add" to add products to cart
3. Use + / - to adjust quantity
4. Click "Chat" to message vendor

**Checkout:**
1. Click "Cart" in header
2. Review items
3. Click "Proceed to Checkout"
4. Enter delivery address
5. Choose payment method
6. Place order

**Track Order:**
1. Click "Orders" in header
2. See live order status
3. Contact delivery partner
4. Rate your experience

## 🐛 Something Not Working?

### Quick Fixes

**Port already in use?**
```bash
npm run dev -- --port 3001
```

**Changes not showing?**
```bash
# Hard refresh
Ctrl/Cmd + Shift + R
```

**Error after installing?**
```bash
# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install
```

**TypeScript errors?**
```bash
# Check types
npx tsc --noEmit
```
